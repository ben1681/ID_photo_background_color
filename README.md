# 证件照背景颜色替换
证件照背景颜色替换；输入一张证件照，指定背景颜色，运行程序，自动替换证件照底色。


#### 1）把证件照的底色替换成红色

```shell
python red_background.py
```

#### 2）把证件照的底色替换成白色

```shell
python while_background.py
```
